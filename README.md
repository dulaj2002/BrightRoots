# BrightRoots

A comprehensive platform for parents to support their children's development from birth to 5 years.

## Features

- Interactive Learning Center with age-appropriate activities
- Parent Discussion Forum
- Expert Consultant Services
- Growth Tracker for milestones
- Articles and Tips from specialists
- FAQ section

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Vite
- React Router
- React i18next for internationalization
